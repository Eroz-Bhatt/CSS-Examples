Holographic Masks

Experimenting with some CSS holographic effects, inspired by this guide from Robb Owen: Holograms, light-leaks and how to build CSS-only shaders.

Note: This effect relies on background-attachment: fixed; which doesn't work as expected on iOS Safari. It also relies on scrolling over the images to see the gradient and mask effect. Scroll down to see it in action!
