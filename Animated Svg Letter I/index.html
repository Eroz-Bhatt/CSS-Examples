Animated SVG Letter "i"

This animation demonstrates how CSS can create drawing effects on SVG paths using stroke properties and animations

How the CSS Animation Works

CSS Properties Breakdown

The animation uses these key CSS properties:

  • stroke-dasharray: 1500; - Creates a dashed pattern where the dash length is 1500px and gap is 1500px
  • stroke-dashoffset: 3000; - Shifts the dash pattern by 3000px (making the path appear invisible)
  • animation: lines 4s infinite; - Animates the dashoffset over 4 seconds, infinitely
  • animation-delay: calc(var(--item) * 1s); - Staggers the animation for each path
  • stroke: var(--color); - Sets the stroke color using CSS variables defined in the SVG

Animation Mechanics

The animation creates a drawing effect through these steps:

  1. Initially, the entire path is visible because stroke-dashoffset is 0
  2. As the animation progresses, the stroke-dashoffset increases to 3000
  3. This offset moves the dash pattern along the path, creating the illusion of the path being drawn
  4. When the offset reaches 3000 (twice the dasharray value), the path becomes invisible
  5. The animation then resets and repeats

Color System

The SVG uses CSS variables for colors:

  • Stem 1: RGB(246, 9, 149)
  • Stem 2: RGB(254, 219, 59)
  • Stem 3: RGB(49, 191, 186)
  • Dot 1: RGB(88, 40, 215)
  • Dot 2: RGB(246, 9, 149)
  • Dot 3: RGB(254, 219, 59)

Staggered Animation

The animation uses a CSS variable (--item) to create delays:

  • Path 1: Delay = 1s × 1 = 1s
  • Path 2: Delay = 1s × 2 = 2s
  • Path 3: Delay = 1s × 3 = 3s

This creates a sequential animation effect where paths appear to be drawn one after another.
